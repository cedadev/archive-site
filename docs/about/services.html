Analysis compute services

Name Also known as Details
Scientific analysis servers sci Interactive compute nodes for developing & testing code
Batch processing cluster lotus ~14,000 core mini-HPC for data analysis processing. Currently dominated by CPU nodes, some GPU nodes coming soon.
Software jaspy Common Python-based software environment and additional components.
JASMIN Notebooks service Provides a web-based interactive Python notebook to experiment with code and data.

Storage services

Name Also known as Details
Home /home/users Home directory for user configuration files, code and other small but important data which need to be backed up.
Group Workspaces GWS Large-scale (1TB to 1PB) but short-term (<5yrs) collaborative storage made available to a group for the duration of a project. GWS manager responsible for organisation of data and controlling access. Comprise storage of different types: disk, tape and object storage.
Transfer Cache XFC Per-user temporary multi-TB storage with automated usage limits, ideal for re-working data brought in from external sources before transferring to GWS tape or disk storage.
Scratch Temporary storage provided for intermediate data when using the LOTUS batch processing cluster.
CEDA Archive Long-term curated data archive of Atmospheric science, Climate model and Earth Observation data. Available read-only throughout JASMIN for easy processing.
Joint Data Migration Application JDMA Elastic Tape Application for copying/moving data between different media, including tape, disk and object storage.

Community Cloud

Name Also known as Details
Cloud portal https://cloud.jasmin.ac.uk Portal for management of cloud tenancies. Allows deployment of virtual machines from catalogue of provided templates.
Managed tenancy Enables communities to have their own "sci" machine(s) over which they have some control.
External tenancy Portions of cloud infrastructure in which projects can build their own bespoke computing infrastructure, from scratch - for example a project website - or using prebuilt templates using JASMIN's Cluster-as-a-Service (CaaS) system.
